# CizaOnline Premium Website

Site Next.js premium pour CizaOnline, avec TailwindCSS et Framer Motion.

## Lancer le projet

```bash
npm install
npm run dev
```

Ouvre ensuite `http://localhost:3000`.

## Stack

- Next.js
- TailwindCSS
- Framer Motion
- Lucide React

## Asset

Le visuel hero a ete genere avec l'outil imagegen et copie dans `public/images/ciza-hero.png`.
